### 功能描述

保存首页告警图配置

### 请求参数

| 字段  | 类型   | 必选 | 描述       |
|-------|--------|------|------------|
| bk_biz_id | int | 是   | 业务ID |
| config | object | 是   | 配置 |

#### config 字段说明
| 字段 | 类型 | 描述 |
|------|------|------|
| name | string | 名称 |
| strategy_ids | list | 策略ID |

### 请求参数示例

```json
{
    "bk_biz_id": 2,
    "config": [{"name": "主机告警", "strategy_ids": [1, 2, 3]}]
}
```

### 响应参数

| 字段 | 类型 | 描述 |
|------|------|------|
| result | bool | 是否成功 |
