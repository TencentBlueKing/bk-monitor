### 功能描述

删除插件


### 请求参数

| 字段       | 类型        | 必填 | 描述                                                                 |
|------------|-----------|------|------------------------------------------------------------------------|
| plugin_ids | list[str] | 是   | 插件ID列表|


### 请求参数示例

```json
{
    "plugin_ids": ["sss_script", "test_plugin"]
}
```

### 响应参数

| 字段    | 类型   | 描述         |
|---------|--------|------------|
| result  | bool   | 请求是否成功 |
| code    | int    | 返回的状态码 |
| message | string | 描述信息     |
| data    | dict   | 结果数据     |

#### data 字段说明

| 字段   | 类型 | 描述                 |
|--------|------|------------------------|
| result | bool | 删除操作是否成功 |


### 响应参数示例

```json
{
  "result": true,
  "code": 200,
  "message": "OK",
  "data": {"result": true}
}
```
