### 功能描述

同步单业务的计算平台RT元信息

### 请求参数

| 字段        | 类型  | 必选 | 描述             |
|-----------|-----|----|----------------|
| bk_biz_id | str | 是  | 业务ID           |
| page      | int | 否  | 页码，默认为1，最小值为1  |
| page_size | int | 否  | 每页数量，默认为0（不分页） |

### 请求参数示例

```json
{
    "bk_biz_id": "2"
}
```

### 响应参数

| 字段      | 类型     | 描述     |
|---------|--------|--------|
| result  | bool   | 请求是否成功 |
| code    | int    | 返回的状态码 |
| message | string | 描述信息   |
| data    | null   | 无返回数据  |

### 响应参数示例

```json
{
    "result": true,
    "code": 200,
    "message": "OK",
    "data": null
}
```
