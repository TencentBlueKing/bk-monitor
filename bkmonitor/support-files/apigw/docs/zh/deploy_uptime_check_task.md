### 功能描述

下发拨测任务


### 请求参数

| 字段        | 类型  | 必选 | 描述     |
|-----------|-----|----|--------|
| task_id   | int | 是  | 拨测任务ID |

### 请求参数示例

```json
{
  "task_id": 10018
}
```

### 响应参数

| 字段      | 类型   | 描述     |
|---------|------|--------|
| result  | bool | 请求是否成功 |
| code    | int  | 返回的状态码 |
| message | str  | 描述信息   |
| data    | str  | 描述信息   |

### 响应参数示例

```json
{
  "result": true,
  "code": 200,
  "message": "OK",
  "data": "success"
}
```
