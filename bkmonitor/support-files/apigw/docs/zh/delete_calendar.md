### 功能描述

删除日历


### 请求参数

| 字段名 | 类型 | 必选 | 描述   |
| ------ | ---- | ---- | ------ |
| int    | Int  | 是   | 日历ID |

### 请求参数示例

```json
{
    "id": 1
}
```

### 响应参数

| 字段名  | 类型   | 描述         |
| ------- | ------ | ------------ |
| result  | Bool   | 请求是否成功 |
| code    | Int    | 返回的状态码 |
| message | String | 描述信息     |
| data    | Null   | 空           |

### 响应参数示例

```json
{
    "message": "OK",
    "code": 200,
    "data": null,
    "result": true
}
```
