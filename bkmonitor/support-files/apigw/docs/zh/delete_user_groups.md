### 功能描述

删除用户组

### 请求参数

| 字段名 | 类型   | 是否必选 | 描述    |
|-----|------|------|-------|
| ids | list | 是    | 通知组ID |

### 请求参数示例

```json
{
  "ids": [123, 456]
}
```

### 响应参数

| 字段      | 类型   | 描述         |
|---------|------|------------|
| result  | bool | 请求是否成功     |
| code    | int  | 返回的状态码     |
| message | str  | 描述信息       |
| data    | null | 删除结果（无返回值） |

### 响应参数示例

```json
{
  "result": true,
  "code": 200,
  "message": "OK",
  "data": null
}
```
