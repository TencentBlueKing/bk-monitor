
### 功能描述

批量删除自定义时序指标分组
删除指定自定义时序数据源下的指标分组，支持批量删除多个分组


### 请求参数

| 字段           | 类型   | 必选 | 描述        |
| -------------- | ------ | ---- | ----------- |
| group_id | int | 是 | 自定义时序数据源 ID |
| scopes | list | 是 | 批量删除的分组列表，至少包含一个分组 |

#### scopes 列表项字段说明

| 字段           | 类型   | 必选 | 描述             |
| -------------- | ------ | ---- |----------------|
| scope_name | string | 是 | 指标分组名，最大长度 255 |

### 请求参数示例

```json
{
	"group_id": 123,
	"scopes": [
		{
			"scope_name": "指标分组名1"
		},
		{
			"scope_name": "指标分组名2"
		}
	]
}
```

### 响应参数

| 字段       | 类型   | 描述         |
| ---------- | ------ | ------------ |
| result     | bool   | 请求是否成功 |
| code       | int    | 返回的状态码 |
| message    | string | 描述信息     |
| data       | null   | 数据，删除成功时为空 |
| request_id | string | 请求ID       |

### 响应参数示例

```json
{
    "message":"OK",
    "code":200,
    "data": null,
    "result":true,
    "request_id":"408233306947415bb1772a86b9536867"
}
```
