### 功能描述

删除处理套餐


### 请求参数

| 字段 | 类型 | 必选 | 描述       |
| ---- | ---- | ---- | ---------- |
| id   | int  | 是   | 处理套餐ID |

### 请求参数示例

```json
{
    "id": 1
}
```

### 响应参数

| 字段    | 类型   | 描述         |
| ------- | ------ | ------------ |
| result  | bool   | 请求是否成功 |
| code    | int    | 返回的状态码 |
| message | str    | 描述信息     |
| data    | dict   | 删除的处理套餐信息 |

#### data字段说明

| 字段 | 类型 | 描述       |
| ---- | ---- | ---------- |
| id   | int  | 处理套餐ID |
| name | str  | 处理套餐名称 |

### 响应参数示例

```json
{
    "message": "OK",
    "code": 200,
    "result": true,
    "data": {
        "id": 1,
        "name": "测试新建处理套餐"
    }
}
```
