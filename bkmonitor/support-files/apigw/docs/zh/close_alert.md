### 功能描述

关闭告警


#### 接口参数

| 字段      | 类型         | 必选 | 描述       |
| --------- | ------------ | ---- | ---------- |
| bk_biz_id | int          | 是   | 业务ID     |
| ids       | list[string] | 是   | 告警ID列表 |
| message   | string       | 否   | 确认信息，默认空字符串   |

#### 示例数据

```json
{
    "ids": [
        "16424876305819837",
        "16424876305819838",
        "16424876305819839",
        "16424876305819840",
        "16424876305819841"
    ],
    "message": "主动关闭",
    "bk_biz_id": 2
}
```

### 响应参数

| 字段    | 类型   | 描述         |
| ------- | ------ | ------------ |
| result  | bool   | 请求是否成功 |
| code    | int    | 返回的状态码 |
| message | string | 描述信息     |
| data    | dict   | 操作结果     |

#### 示例数据

```json
{
    "message": "OK",
    "code": 200,
    "data": {
        "alerts_close_success": [
            "16424876305819837",
            "16424876305819838"
        ],
        "alerts_not_exist": [
            "16424876305819839"
        ],
        "alerts_already_end": [
            "16424876305819840",
            "16424876305819841"
        ]
    },
    "result": true
}
```
