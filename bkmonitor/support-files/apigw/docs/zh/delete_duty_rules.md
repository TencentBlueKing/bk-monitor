### 功能描述

批量删除轮值规则


#### 接口参数

| 字段         | 类型        | 必选 | 描述     |
|------------|-----------|----|--------|
| bk_biz_ids | list[int] | 是  | 业务ID   |
| ids        | list[int] | 是  | 轮值规则ID |

#### 请求示例

```json
{
  "bk_biz_ids": [
    2,
    3
  ],
  "ids": [
    87,
    88,
    89
  ]
}
```

### 响应参数

| 字段      | 类型   | 描述     |
|---------|------|--------|
| resul   | bool | 请求是否成功 |
| code    | int  | 返回的状态码 |
| message | str  | 描述信息   |
| data    | dict | 结果     |

#### data.ids

| 字段  | 类型        | 描述     |
|-----|-----------|--------|
| ids | list[int] | 轮值规则ID |

#### 响应示例

```json
{
  "result": true,
  "code": 200,
  "message": "OK",
  "data": {
    "ids": [
      87,
      88,
      89
    ]
  }
}
```
