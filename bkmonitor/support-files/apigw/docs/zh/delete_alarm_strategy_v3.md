### 功能描述

删除告警策略(迁移)

### 请求参数

| 字段名       | 类型        | 是否必选 | 描述             |
|-----------|-----------|------|----------------|
| bk_biz_id | int       | 是    | 业务ID           |
| id        | int       | 否    | 策略ID（与ids二选一）  |
| ids       | list[int] | 否    | 策略ID列表（与id二选一） |

### 请求参数示例

```json
{
  "bk_biz_id": 2,
  "id": 123
}
```

或

```json
{
  "bk_biz_id": 2,
  "ids": [123, 456]
}
```

### 响应参数

| 字段名     | 类型   | 描述         |
|---------|------|------------|
| result  | bool | 请求是否成功     |
| code    | int  | 返回的状态码     |
| message | str  | 描述信息       |
| data    | null | 删除结果（无返回值） |

### 响应参数示例

```json
{
  "result": true,
  "code": 200,
  "message": "OK",
  "data": null
}
```
