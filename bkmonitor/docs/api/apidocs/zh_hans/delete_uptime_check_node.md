### 功能描述

删除拨测节点

### 请求参数

{{ common_args_desc }}

#### 接口参数

| 字段             | 类型    | 必选 | 描述                                |
|----------------|-------|-----|-----------------------------------|
| bk_biz_id      | int   | 是  | 业务ID                              |
| node_id        | int   | 是  | 要删除的拨测节点ID                              |

#### 示例数据

```json
{
    "bk_biz_id": 2,
    "node_id": 10255
}
```

### 响应参数

| 字段         | 类型  | 描述 |
|:-----------|-----|----|
| id         | int   | 删除的拨测节点ID |

#### 示例数据
```json
{
    "result": true,
    "code": 200,
    "message": "OK",
    "data": {
        "id": 10255
    }
}
```
