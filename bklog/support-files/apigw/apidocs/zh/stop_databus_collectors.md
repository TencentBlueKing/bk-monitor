## 功能描述

停止普通日志采集项，停止后目标节点不再采集日志。

## 请求参数

### 鉴权头

| 参数名称    | 参数类型 | 必须 | 参数说明     |
| ----------- | -------- | ---- | ------------ |
| app_code    | string   | 是   | 蓝鲸应用ID   |
| app_secret  | string   | 是   | 蓝鲸应用秘钥 |
| bk_username | string   | 是   | 用户名称     |

鉴权信息通过请求头 `X-Bkapi-Authorization` 传递，取值为上述字段构成的 JSON 字符串。

### 路径参数

| 字段                | 类型 | 必选 | 描述      |
| ------------------- | ---- | ---- | --------- |
| collector_config_id | int  | 是   | 采集项 ID |

### 参数列表

| 字段              | 类型 | 必选 | 描述                                 |
| ----------------- | ---- | ---- | ------------------------------------ |
| is_stop_index_set | bool | 否   | 是否同时停用索引集，默认 `true`      |

## 请求参数示例

```json
{
    "is_stop_index_set": true
}
```

## 返回结果示例

```json
{
    "result": true,
    "code": 0,
    "message": "",
    "data": null
}
```
