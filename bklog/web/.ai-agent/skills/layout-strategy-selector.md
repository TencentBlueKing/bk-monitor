# Skill: Layout Strategy Selector

Select graph layout strategy only after clarifying graph scale and interaction constraints.

Selection dimensions:
- automatic layout vs manual positioning
- DAG, tree, topology or free graph
- incremental updates
- user drag and persisted coordinates
- performance under large node and edge counts

Output:
- selected layout strategy
- rejected strategies and tradeoffs
- integration boundary with rendering runtime

Required artifacts:
- layout_strategy
