# Skill: Runtime Evolution Predictor

Predict graph runtime evolution before locking implementation boundaries.

Focus:
- asynchronous graph data loading
- streaming node or edge updates
- retry and cancellation
- command history
- layout recomputation
- rendering performance

Output:
- likely graph runtime extensions
- stable extension points
- risks of coupling layout, data fetching and rendering

Required artifacts:
- runtime_extensions
- extension_points
